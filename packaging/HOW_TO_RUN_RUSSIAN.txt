========================================================================
  TradingSearch — поиск похожих трейдинговых графиков по фрагменту
========================================================================


КАК ЗАПУСТИТЬ
-------------

1.  Распакуйте присланный ZIP в любую папку. Лучше в "Документы",
    а НЕ в "Program Files" (там нужны права администратора).

    Например:    C:\TradingSearch\

2.  Откройте папку TradingSearch и сделайте двойной клик по файлу

        TradingSearch.exe

    При первом запуске Windows Defender может спросить разрешение —
    выберите "Подробнее" → "Выполнить в любом случае".

3.  Программа запускается 10-15 секунд (загрузка ИИ-модели).
    Когда в правом нижнем углу появится статус "ИИ-движок: готов" —
    можно работать.


КАК ПОЛЬЗОВАТЬСЯ
----------------

Базовый сценарий:

  A.  Один раз: добавить свои графики в базу
      - кнопка "Импорт папки" → выбрать папку с PNG/JPG графиками
      - дождаться окончания индексирования (10-15 графиков в секунду)

  B.  Каждый раз когда нужен поиск:
      1) сделать скриншот фрагмента (клавиша Print Screen) ИЛИ
         нажать "Открыть" и выбрать файл-фрагмент
      2) (необязательно) выделить нужный участок мышью прямо на картинке
      3) выбрать режим: Умный / Точный / Похожие
      4) нажать "Найти" (или просто Enter)

  C.  Чтобы повторить поиск с другим режимом — просто переключите
      кнопку режима и снова нажмите "Найти". Заново вставлять не нужно.

  D.  Кнопка "База графиков" показывает ВСЕ загруженные графики
      сеткой с миниатюрами. Двойной клик — открыть для поиска.


РЕЖИМЫ ПОИСКА
-------------

  Умный   — рекомендую по умолчанию. ИИ отбирает 200 похожих графиков,
            потом ORB проверяет каждый. Универсальный режим.

  Точный  — полный ORB-перебор по всей базе. Самый точный для пиксельно
            идентичных кропов, но медленнее (10-15 секунд).

  Похожие — только ИИ-поиск, без проверки геометрии. Самый быстрый.
            Ищет визуально похожие графики даже в другом стиле
            (другие цвета, другой фон).


ГДЕ ЛЕЖАТ ДАННЫЕ
-----------------

Импортированные графики, дескрипторы и эмбеддинги хранятся в

  %LOCALAPPDATA%\TradingSearch\data\

То есть   C:\Users\<ваше_имя>\AppData\Local\TradingSearch\data\

Если перенесёте эту папку на другой компьютер — вся база переедет
с вами, переиндексировать заново не надо.

Если хотите начать с нуля — закройте программу и удалите эту папку,
при следующем запуске она создастся пустой.


СИСТЕМНЫЕ ТРЕБОВАНИЯ
--------------------

  - Windows 10/11 64-бит
  - 4 ГБ оперативной памяти (8 ГБ при базе >5000 графиков)
  - ~2 ГБ свободного места на диске для самой программы
    + ~100 МБ на каждые 1000 графиков в базе


ПРОБЛЕМЫ
--------

  - "ИИ-движок: загружается…" висит больше минуты:
    программа всё ещё инициализируется, подождите.

  - Поиск ничего не находит:
    проверьте что база не пустая ("База графиков" покажет содержимое).
    Если база пустая — сначала "Импорт папки".

  - Ctrl+V не вставляет: убедитесь что в буфере обмена именно
    изображение (например, после Print Screen), а не текст.
